## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/vanarok/ui-perfect-template.git
cd ui-perfect-template
npm i
npm run dev
```
